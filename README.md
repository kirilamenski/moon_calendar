# About
Shows moon phases.

<img src="http://i.imgur.com/bPssEgs.png" />

# Usage
```
git clone https://github.com/kirilamenski/moon_calendar.git
cd ~./moon_calendar
cargo build
cargo run
```

| Keyboard | Description |
| ------------- | ------------- |
| left/right arrows  | change day  |
| up/down arrows  | change month  |
